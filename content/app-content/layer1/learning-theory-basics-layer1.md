## Page 1 — Introduction to Learning Theory
**Page Type:** A (Overview)

### Purpose
This page provides an introduction to fundamental learning theories and their application.

### Overview
Learning theory examines how people acquire, process, and retain knowledge. Multiple theoretical frameworks have been developed to explain and optimize learning processes.

### Key Learning Approaches
Common learning theories include:
- behaviorism and reinforcement
- cognitive processing models
- constructivism and social learning
- experiential learning frameworks

### Scope of This Module
This module introduces major learning theories and their practical applications in educational settings.

## Page 2 — Behaviorist Learning Theory
**Page Type:** B (Concept)  
**Sensitivity:** Low  
**Depth:** Overview

### Summary
- Behaviorism focuses on observable behaviors and environmental stimuli.
- Learning occurs through conditioning and reinforcement.
- This approach emphasizes practice and repetition.

### Core Concepts
Key behaviorist principles:
- **Stimulus-Response:** Learning as association between stimuli and responses.
- **Reinforcement:** Behaviors followed by positive outcomes are strengthened.
- **Practice:** Repetition consolidates learning.

### Application in Education
Behaviorist principles inform:
- skill practice and drills
- reward systems and feedback
- structured learning sequences

## Page 3 — Cognitive Learning Theory
**Page Type:** B (Concept)  
**Sensitivity:** Low  
**Depth:** Overview

### Summary
- Cognitive theory focuses on mental processes in learning.
- Learning involves information processing and schema development.
- Understanding how people think informs instruction.

### Key Concepts
- **Information Processing:** How learners encode, store, and retrieve information.
- **Schema:** Mental frameworks that organize knowledge.
- **Metacognition:** Awareness and control of one's own thinking.

### Educational Implications
Cognitive theory supports:
- strategies for memory and recall
- problem-solving instruction
- self-regulated learning
